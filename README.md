# Bot-Face
Automate Facebook-Messenger With Python

# Installation :
python install.py

# Menu :
1) MESSAGE
2) IMAGE
3) FILE
4) CHANGE NICKNAME
5) BLOCK USER
6) DELETE MESSAGE
7) FETCH USERS
8) REMOVE FRIEND
00) LOGOUT & EXIT
